Getting started

My profile

Your profile is the core and most important part of Connect. So you should decorate it to the fullest (or as much as you can)!

To do this, we've added a BBCode and two classes to the site. Let's start with two simple classes ^w^.

Classes

Classes in CSS are used to apply styling to multiple HTML elements that share the same class attribute. Classes allow you to group elements together and style them uniformly.

Example usage of classes "title" and "avatar":

.title {
font-size: 24px;
font-weight: bold;
color: darkgrey;
}

.avatar {
width: 100px;
height: 100px;
border: 5px double red;
}

Want more customization? So use classes .wrapper and .online to customize your page:

.wrapper {
border: 4px ridge yellow; /* More info at MDN web docs*/
max-height: 765px;
overflow: auto; /* Adds a scrollbar to your page */
}
.online {
border: thick double #32a1ce;
}
.status-online {
color: lightgreen;
}
.status-offline {
color: firebrick; /* More colors at W3schools*/
}
.lastseen {
color: rgba(0, 0, 0, .5); /* More info at WebFX */
}

Don't forget about .footer1 and .footer2, these two classes will allow you to change the footer of the page, you can hide it or colorize it:

.footer1 { /* Website version in the page footer */
color: green;
}
.footer2 { /* Banner in page footer */
display: none;
}

Recommendations

You can look into the source code of the page by pressing F12 or Right Mouse Button -> Explore, there you can modify styles directly and see the changes in real time, then you can copy and apply the styles to your page.

You can also change the styles of html elements such as body, for example you can set background-color: black; and color: white; you can do the same with any other styles.
